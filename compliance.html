US Privacy Compliance — automatic opt-out (GPC)

The MediaJel tag will detect US privacy opt-out signals and refuse to initialize tracking — on every site, with no per-site setup.

What we're building & why

US privacy laws (California's CCPA/CPRA, plus Colorado and Connecticut) recognize a browser-level opt-out signal a user can set once. We honor it inside the tag itself, so the protection applies everywhere the tag is embedded — no consent banner, no cookie, no work from the sites we're on.

The exact tracker changes

1. Create — src/shared/utils/privacy-opt-out.ts (pure signal detection)

/** Detects US privacy opt-out signals (GPC + legacy DNT) at the browser level. */
export const isUsPrivacyOptOut = (): boolean => {
  const gpcOptOut = navigator.globalPrivacyControl === true;

  const dntRaw = navigator.doNotTrack || navigator.msDoNotTrack || window.doNotTrack;
  const dntOptOut = dntRaw === "1" || dntRaw === "yes";

  return gpcOptOut || dntOptOut;
};

export default isUsPrivacyOptOut;

2. Modify — src/shared/interface.ts (global type augmentation)

declare global {
  interface Navigator {
    globalPrivacyControl?: boolean;
    msDoNotTrack?: string | null;
  }
  interface Window {
    doNotTrack?: string | null;
    // ...existing fields unchanged...
  }
}

3. Modify — src/index.ts (gate, right after getContext(), before any network call)

const context: QueryStringContext = getContext();

// US privacy opt-out gate — honor GPC / DNT before any tracking or network activity.
// Hard no-track: return before adapters, Snowplow, custom tags, or appId tags load.
if (isUsPrivacyOptOut()) {
  logger.debug("US privacy opt-out detected (GPC/DNT). Tracker will not initialize.");
  return;
}

await getCustomTags();
await getAppIdTags(context.appId);

This is stronger than Snowplow's respectDoNotTrack (which only gates sending, and only reads DNT, not GPC). Here the tracker never starts, so nothing is sent and no cookies are set.

How it's verified

Automated (Cypress E2E)

Stub the signal before the tag loads; assert the collector is never hit.

cy.intercept("POST", "**/analytics/track", { statusCode: 200, body: {} }).as("track");

cy.visit("http://localhost:1234/", {
  onBeforeLoad(win) {
    Object.defineProperty(win.navigator, "globalPrivacyControl", { value: true, configurable: true });
  },
});
cy.wait(3000);
cy.get("@track.all").should("have.length", 0);   // tag suppressed

Run:

npm run check
npx start-server-and-test 'npm run bootstrap-test-server' http://localhost:1234 \
  'npx cypress run --spec cypress/e2e/privacy-opt-out.spec.cy.ts'

Manual (real signal)

  • npm start, then open the demo in Brave (GPC on by default) or Firefox (“Tell websites not to sell or share my data”).
  • DevTools → Network, filter track: no POST …/analytics/track; console logs the opt-out.
  • Turn the signal off, reload: collector requests return.
