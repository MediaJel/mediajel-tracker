Constructor

# new createSnowplowStandardTracker(input) → {SnowplowTracker}

Function factory that creates a Snowplow Standard tracker instance. The methods included in the tracker are specific to the Snowplow Standard tracker also known as "v3" in the snowplow docs
Parameters:
Name Type Description
input SnowplowTrackerInput Input object for the Snowplow tracker
appId SnowplowTrackerInput.appId The unique identifier for the client that is sending the event
collector SnowplowTrackerInput.collector A snowplow collector url
event SnowplowTrackerInput.event The event that the tracker is configured for
environment SnowplowTrackerInput.environment The environment template selected
version SnowplowTrackerInput.version The version of the tracker
